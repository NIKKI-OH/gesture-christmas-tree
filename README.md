# Gesture Christmas Tree 🎄✨

用手势控制的3D粒子圣诞树！张开手掌观看粒子漂浮，握拳变成粉色圣诞树。

## 如何体验
1. 点击下方链接
2. 允许摄像头权限
3. 用手势控制效果

[点击体验](https://NIKKI-OH.github.io/gesture-christmas-tree/)

## 手势控制
- ✋ 张开手掌：粒子自由漂浮
- ✊ 握拳：形成圣诞树
- 🤏 捏合：调整大小

## 技术栈
- Three.js - 3D渲染
- MediaPipe - 手势识别
- Vite - 构建工具
